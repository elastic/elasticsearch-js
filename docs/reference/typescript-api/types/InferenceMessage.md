# InferenceMessage

## Interface

### Properties

| Property | Type | Description |
|----------|------|-------------|
| `content?` | [`InferenceMessageContent`](InferenceMessageContent.md) | The content of the message.

String example:
```
{
   "content": "Some string"
}
```

Object example:
```
{
  "content": [
      {
       "text": "Some text",
       "type": "text"
      }
   ]
}
``` |
| `role` | `string` | The role of the message author. Valid values are `user`, `assistant`, `system`, and `tool`. |
| `tool_call_id?` | [`Id`](Id.md) | Only for `tool` role messages. The tool call that this message is responding to. |
| `tool_calls?` | `InferenceToolCall[]` | Only for `assistant` role messages. The tool calls generated by the model. If it's specified, the `content` field is optional.
Example:
```
{
  "tool_calls": [
      {
          "id": "call_KcAjWtAww20AihPHphUh46Gd",
          "type": "function",
          "function": {
              "name": "get_current_weather",
              "arguments": "{\"location\":\"Boston, MA\"}"
          }
      }
  ]
}
``` |
