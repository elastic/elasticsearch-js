# IndicesMappingLimitSettingsDepth

## Interface

### Properties

| Property | Type | Description |
|----------|------|-------------|
| `limit?` | `long` | The maximum depth for a field, which is measured as the number of inner objects. For instance, if all fields are defined
at the root object level, then the depth is 1. If there is one object mapping, then the depth is 2, etc. |
