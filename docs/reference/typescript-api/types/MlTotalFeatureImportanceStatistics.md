# MlTotalFeatureImportanceStatistics

## Interface

### Properties

| Property | Type | Description |
|----------|------|-------------|
| `mean_magnitude` | `double` | The average magnitude of this feature across all the training data. This value is the average of the absolute values of the importance for this feature. |
| `max` | `integer` | The maximum importance value across all the training data for this feature. |
| `min` | `integer` | The minimum importance value across all the training data for this feature. |
