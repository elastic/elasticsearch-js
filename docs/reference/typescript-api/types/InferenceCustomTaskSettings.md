# InferenceCustomTaskSettings

## Interface

### Properties

| Property | Type | Description |
|----------|------|-------------|
| `parameters?` | `any` | Specifies parameters that are required to run the custom service. The parameters depend on the model your custom service uses.
For example:
```
"task_settings":{
  "parameters":{
    "input_type":"query",
    "return_token":true
  }
}
``` |
