# WatcherExecutionThreadPool

## Interface

### Properties

| Property | Type | Description |
|----------|------|-------------|
| `max_size` | `long` | The largest size of the execution thread pool, which indicates the largest number of concurrent running watches. |
| `queue_size` | `long` | The number of watches that were triggered and are currently queued. |
