# AnalysisLengthTokenFilter

## Interface

### Properties

| Property | Type | Description |
|----------|------|-------------|
| `type` | `'length'` | - |
| `max?` | `integer` | Maximum character length of a token. Longer tokens are excluded from the output. Defaults to `Integer.MAX_VALUE`, which is `2^31-1` or `2147483647`. |
| `min?` | `integer` | Minimum character length of a token. Shorter tokens are excluded from the output. Defaults to `0`. |
