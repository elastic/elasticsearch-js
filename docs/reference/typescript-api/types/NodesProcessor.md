# NodesProcessor

## Interface

### Properties

| Property | Type | Description |
|----------|------|-------------|
| `count?` | `long` | Number of documents transformed by the processor. |
| `current?` | `long` | Number of documents currently being transformed by the processor. |
| `failed?` | `long` | Number of failed operations for the processor. |
| `time_in_millis?` | `DurationValue<UnitMillis>` | Time, in milliseconds, spent by the processor transforming documents. |
