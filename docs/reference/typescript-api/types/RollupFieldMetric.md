# RollupFieldMetric

## Interface

### Properties

| Property | Type | Description |
|----------|------|-------------|
| `field` | [`Field`](Field.md) | The field to collect metrics for. This must be a numeric of some kind. |
| `metrics` | `RollupMetric[]` | An array of metrics to collect for the field. At least one metric must be configured. |
