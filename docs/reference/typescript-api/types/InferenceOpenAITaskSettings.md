# InferenceOpenAITaskSettings

## Interface

### Properties

| Property | Type | Description |
|----------|------|-------------|
| `user?` | `string` | For a `completion` or `text_embedding` task, specify the user issuing the request.
This information can be used for abuse detection. |
| `headers?` | `any` | Specifies custom HTTP header parameters.
For example:
```
"headers":{
  "Custom-Header": "Some-Value",
  "Another-Custom-Header": "Another-Value"
}
``` |
