# SecurityDeleteServiceTokenResponse

## Interface

### Properties

| Property | Type | Description |
|----------|------|-------------|
| `found` | `boolean` | If the service account token is successfully deleted, the request returns `{"found": true}`.
Otherwise, the response will have status code 404 and `found` is set to `false`. |
