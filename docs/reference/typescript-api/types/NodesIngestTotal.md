# NodesIngestTotal

## Interface

### Properties

| Property | Type | Description |
|----------|------|-------------|
| `count` | `long` | Total number of documents ingested during the lifetime of this node. |
| `current` | `long` | Total number of documents currently being ingested. |
| `failed` | `long` | Total number of failed ingest operations during the lifetime of this node. |
| `time_in_millis` | `DurationValue<UnitMillis>` | Total time, in milliseconds, spent preprocessing ingest documents during the lifetime of this node. |
