# TypeScript API Reference

Complete TypeScript API reference for the Elasticsearch JavaScript client.

## Documentation Structure

- **[index.md](./index.md)** - Main API reference index
- **[client.md](./client.md)** - Client class documentation
- **[apis/](./apis/)** - Individual API method documentation
- **[types/](./types/)** - TypeScript type definitions
- **[helpers.md](./helpers.md)** - Helper utilities
- **[transport.md](./transport.md)** - Transport layer

## Key Features

- **Comprehensive**: Every API method and type documented
- **Organized**: Separate file per API and type for easy navigation
- **Cross-referenced**: Links between related APIs and types
- **TypeScript-first**: Generated from TypeScript source code
